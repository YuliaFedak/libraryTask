Backend:

1. Clone the repository:
   ```bash
   git clone https://github.com/YuliaFedak/libraryTasko.git

   cd server

2. Install dependencies:
    ```bash
    npm install

3. To start the application:
    ```bash
    npm run startLdev

2. Install dependencies:
